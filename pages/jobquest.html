About the Project

JobQuest is a comprehensive mobile application that connects job seekers with employers. The app provides a seamless platform for job searching, application management, and recruitment processes. With an intuitive interface and powerful features, JobQuest simplifies the job hunting experience for both job seekers and employers.

The application features role-based dashboards for job seekers and employers, advanced job search filters, profile management, real-time notifications, and secure authentication. Built with Flutter and Firebase, JobQuest delivers a smooth, responsive experience across all devices.

Key Features

Secure Authentication

Multi-factor authentication with email verification and password recovery.

Dual Role System

Separate experiences for job seekers and employers with tailored dashboards.

Advanced Search

Filter jobs by location, salary, industry, and experience level.

Save Jobs

Bookmark interesting positions for later review and application.

Real-time Notifications

Instant updates on application status and new job postings.

Profile Management

Create and edit professional profiles with resume upload.
