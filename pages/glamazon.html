About the Project

Glamazon is a comprehensive mobile application designed to revolutionize the salon booking experience. Built with Flutter and Firebase, this app provides a seamless platform for customers to discover, book, and manage salon appointments while enabling salon owners to efficiently manage their business operations.

The application features a beautiful, intuitive interface with role-based access control, allowing both customers and salon owners to have tailored experiences. From browsing available services to managing bookings and processing payments, Glamazon handles the entire salon appointment lifecycle.

Key Features

User Authentication

Secure sign-up and login system with email verification and social media integration.

Easy Booking

Intuitive booking interface allowing users to schedule appointments at their convenience.

Role-Based Access

Separate interfaces for customers and salon owners with appropriate permissions.

Salon Discovery

Browse and search for salons based on location, services, and ratings.

Push Notifications

Real-time notifications for booking confirmations, reminders, and updates.

Business Analytics

Comprehensive dashboard for salon owners to track bookings and revenue.
